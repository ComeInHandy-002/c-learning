### Day 4-5：指针与数组（12h）
**面试常考核心题**：
```
1. 数组名 vs &数组名
   int a[5];
   sizeof(a) = 20 / sizeof(a+0) = 4或8 / sizeof(&a) = 4或8

2. 指针数组 vs 数组指针 vs 函数指针
   int *arr[10];        ← 指针数组
   int (*arr)[10];      ← 数组指针
   int (*p)();          ← 函数指针
   int *p();            ← 返回指针的函数

3. 二维数组 int a[3][4] 的 a、a[0]、&a[0] 各是什么类型
```
**综合练习**：用指针版本实现 strlen/strcpy/strcat/strcmp，不用任何下标

### Day 6-7：函数指针 + 回顾（12h）
```
Day 6：函数指针 + 用函数指针实现简易计算器（加减乘除），push
Day 7：韦东山《C语言加强》指针部分 + 复习 Week 1 笔记
```

## 📅 Week 2（Day 8-14）：内存 + 字符串

### Day 8-9：内存布局（12h）
**内存五区必背**：
```
代码区(Text) / 只读数据(RO data) / 已初始化全局(RW data)
未初始化全局(BSS) / 栈(Stack) / 堆(Heap)
```
**实践**：写程序打印各种变量地址，观察分布

### Day 10：内存对齐（6h）
**3 条规则**：
```
1. 成员偏移量必须是成员大小的整数倍
2. struct 整体大小是最大成员大小的整数倍
3. 嵌套 struct 用嵌套 struct 的大小作为"成员大小"
```
**练习**：口算 sizeof 给各种 struct，再用代码验证

### Day 11-12：字符串（12h）
**核心**：
```
字符串本质：以 \0 结尾的 char 数组
char *p = "hello";   // 只读区，不能改！
char a[] = "hello";  // 栈，能改
```
**手写实现**（面试常考）：my_strlen / my_strcpy / my_strcat / my_strcmp

## 📅 Week 3（Day 15-21）：进阶应用 + 小项目

### Day 15：多文件编程 + Makefile 入门（6h）
```
多文件：.h 和 .c 关系 / extern / static 限作用域 / #ifndef 防包含
编译流程：预处理→编译→汇编→链接
Makefile：变量 / 规则 / 通用规则(%.o: %.c) / 伪目标(.PHONY)
```

### Day 16-17：动态内存分配（12h）

```

### Day 18-19：链表（12h）

单链表创建/遍历/插入/删除/反转（嵌入式 RTOS、内核链表基础）

```

### Day 20：struct/union/enum + 位操作（6h）
**位操作必须熟练（嵌入式高频）**：
```
置位：reg |= (1 << n)

取反：reg ^= (1 << n)
判断：if (reg & (1 << n))
大小端判断（用 union 实现）
```
